# Random Dog!

## Description

A fun little app that brings joy by displaying a random dog pic at button click.

- This is an exercise in using AJAX and XMLHttpRequest instead of fetch

## Usage

Step 1: Load the site.

Step 2: Click the button.

Step 3: Smile.
